# Startup

Run this command to set up the Syntharra marketing system from scratch.

## Step 1: Create the Folder Structure

Create all directories if they don't exist:

```
syntharra-marketing/
├── prospects/
├── outreach/templates/
├── outreach/sent/
├── outreach/results/
├── content/blog/
├── content/social/
├── content/video/
├── content/email/
├── reports/
├── assets/
└── config/
```

## Step 2: Create Config Files

### `config/target-cities.md`
List the top 20 HVAC markets with notes on which have been prospected:

Priority 1 (start here): Phoenix AZ, Houston TX, Dallas TX, Miami FL, Tampa FL
Priority 2: Atlanta GA, Las Vegas NV, San Antonio TX, Orlando FL, Charlotte NC
Priority 3: Jacksonville FL, Austin TX, Raleigh NC, Nashville TN, Memphis TN
Priority 4: Oklahoma City OK, Tucson AZ, Sacramento CA, San Diego CA, Bakersfield CA

### `config/competitors.md`
Research and list 5-8 direct competitors with their websites, target markets, and visible pricing.

### `config/brand-voice.md`
Create the brand voice guide:
- Professional but human
- Confident without being pushy
- Specific over generic
- No buzzwords (no leverage, synergy, revolutionize, game-changer)
- Talk like someone who understands the trades, not a marketing department

### `config/kpi-targets.md`
Set the initial targets:
- Weeks 1-4: 10-15 qualified leads/month
- Weeks 5-8: 25-35 qualified leads/month
- Weeks 9-12: 50+ qualified leads/month

### `reports/lead-tracker.csv`
Create the master lead tracking file:
date,name,company,email,phone,source,score,status,notes

## Step 3: First Prospector Batch

Run the Prospector skill for the first batch of 50 prospects from Priority 1 cities (Phoenix and Houston).

## Step 4: Draft Outreach Templates

Create the first 3-email cold outreach sequence in `outreach/templates/sequence-v1.md`.

## Step 5: Plan First Week's Content

Create `content/calendar.md` with the first 2 weeks planned:
- 4 blog articles assigned to topics and keywords
- 10 social posts assigned to platforms and themes
- 3-5 video scripts with hooks and formats
- 1 newsletter (if enough leads exist)

## Step 6: Report

Summarize what's been set up and what the first week's execution plan looks like. Include:
- Number of prospects found and scored
- Preview of outreach sequence
- Content calendar overview
- What Dan needs to do (record videos, set up social accounts)
- Anything that's blocked

## What Dan Needs Before You Start

Check these exist. If not, tell Dan what he needs to set up:

- [ ] Facebook Business Page for Syntharra
- [ ] LinkedIn Company Page for Syntharra
- [ ] X (Twitter) account for Syntharra
- [ ] TikTok Business account for Syntharra
- [ ] Instagram Business account for Syntharra
- [ ] YouTube channel for Syntharra
- [ ] CapCut (free) installed for video captions
- [ ] Phone/camera ready for recording short videos

That's it. No expensive tools. No ad budget. Just social accounts and a phone.
