# Report

Generate a performance snapshot. Pull data from all agent output folders and produce a concise report.

## Output

Print to screen (don't save to file — this is for quick checks).

## What to Include

1. **Lead Score:** Qualified leads this month vs. 50/month target
2. **Channel Breakdown:** Leads by source (outreach, blog, social, video, calculator, quiz)
3. **Outreach Health:** Email open rate, click rate, reply rate, demo bookings
4. **Content Health:** Blog traffic trend, top-performing posts, social engagement
5. **Video Health:** Views, engagement, profile visits across TikTok/Reels/Shorts
6. **Funnel Health:** Website visits → leads → demo bookings conversion rate
7. **Top Recommendation:** The single most impactful thing to do right now

Keep it under 300 words. Dan wants a quick read, not a report to read.
