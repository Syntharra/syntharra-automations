## To update phone number
Find `+1 (000) 000-0000` in `syntharra-signature-PASTE-THIS.html` and replace.

## How to apply
1. Open the relevant `.html` file in Chrome > Request Desktop Site
2. Select All (Ctrl+A) > Copy (Ctrl+C)
3. Paste into Gmail signature settings for the corresponding email address
4. Upload the matching profile picture PNG to that Google account's profile photo
