# Weekly Plan

Generate the marketing plan for this week. Read the latest `reports/weekly-intelligence.md` first. If it doesn't exist yet, use default priorities.

## Output

Save to `content/calendar.md` (overwrite the upcoming week section).

## What to Include

For each day of the week, list:
1. **Prospector tasks** — which cities, how many prospects
2. **Outreach tasks** — which batches to send, any template changes
3. **Content Engine tasks** — which blog to write, which social posts to publish
4. **Video Content tasks** — which scripts are ready, which need recording
5. **Conversion Optimizer tasks** — any A/B tests to launch or check
6. **Anything Dan needs to do** — record videos, approve content, set up accounts

## Priority Rules

1. If Intelligence says something specific, follow it
2. If no Intelligence brief exists yet, prioritize: prospecting > outreach > content > video
3. Always maintain minimum cadence: 2 blogs, 5 social posts, 3 video scripts per week
4. Never skip the Friday outreach batch — it catches weekend readers
