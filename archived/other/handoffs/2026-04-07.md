## Lessons logged this session
- Don't trust an external image URL exists just because the file is in the repo — render the email in a real browser and verify before claiming "fixed". Inline HTML wordmarks beat hosted PNG wordmarks for guaranteed rendering across all email clients.
- When patching JSON payloads with embedded HTML, the strings are JSON-escaped — match against the escaped form, not the raw HTML.
- n8n MCP credential strip bug: use the raw REST API (`PUT /workflows/{id}` then `POST /workflows/{id}/activate`) for any workflow update that touches credentials.
