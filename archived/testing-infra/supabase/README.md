# Supabase Migrations

All database migrations for the Syntharra platform.

## Tables

- hvac_standard_agent — client config for HVAC Standard agents
- hvac_call_log — call logs with lead scoring
- stripe_payment_data — Stripe checkout records
- client_subscriptions — subscription billing tracking
- billing_cycles — monthly usage tracking
- overage_charges — overage billing records
- agreement_signatures — e-signature records
