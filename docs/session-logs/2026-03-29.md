## Root Causes Diagnosed

### SMTP Connection Timeout
Railway blocks all outbound SMTP ports (25, 465, 587, 2525). Must use HTTP REST APIs for email on Railway. Fixed permanently with SMTP2GO REST API.

### n8n Monitor Returning null
alertManager initialised nodemailer transport on startup. When checkN8n() tried to send an alert, SMTP timeout crashed the entire function silently inside safeRun(). Removing nodemailer fixed it.

### 25x HVAC Premium Onboarding Ghost Errors
Ops monitor infrastructure.js v2.0 was using POST (not HEAD) on webhook URLs. Fired every 5 min → triggered real workflow with empty `{}` body → "Credentials not found" error (side effect of empty payload). Fix already committed at 20:38 UTC. Not a real credential issue.

## Key Rules to Remember
- Railway = no SMTP. Always use REST/HTTP APIs for email.
- Webhook health checks = HEAD only, never POST.
- If a monitor shows healthy:null, check alertManager for transport errors first.
- Always verify full method contents before str_replace edits to avoid duplicate declarations.
